CSS(Cascading Style Sheets)

        마크업 언어(HTML)가 실제로 화면에
        표시(모양, 글씨체, 색상, 크기...)되는 방법을 기술하는 언어.

CSS Selector(선택자)

        현재 html 문서 내에서 특정한 요소를 선택하는 문자 또는 문자열.
        
        == 스타일을 적용하고자 하는 요소를 선택하는 방법

태그 선택자

        현재 HTML 문서 내에 같은 태그를 선택하는 선택자.

        [작성법]

        태그명 {
            CSS 코드;
        }

        ex)

        pre (
            font size : 50px;
        )

        HTML 문서 내에서 id 속성값이 일치하는 요소를 선택하는 선택지.

        [작성법]

        #id속성값{
            css 코드;
        }
        #== id
id 속성값은 하나의 html문서 안에서 절대 중복되면 안됨
  1. id 선택자 테스트 1
  2. id 선택자 테스트 2
  3. id 선택자 테스트 3
  4. id 선택자 테스트 4
  5. id 선택자 테스트 5

    6. class 선택자

            html 문서내에서 같은 class 속성값을 가지는 태그를 모두 선택하는 선택자

        [작성법]

        .class속성값 {css 코드;}
